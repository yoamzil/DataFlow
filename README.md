# DataFlow

A client management web app built for **Chaabi Cash** to streamline customer data handling and internal workflows.

Built with Next.js 15, Prisma, and PostgreSQL — featuring authentication, form validation, and a clean dashboard UI.

---

## Features

- 🔐 Authentication with hashed passwords (bcryptjs)
- 📋 Client data management — create, view, update, and delete records
- ✅ Form validation with react-hook-form and Zod schemas
- 🎞️ Smooth transitions with Framer Motion
- 🔔 Toast notifications via react-hot-toast
- 🗄️ SQLite database with Prisma ORM

---

## Tech Stack

| Layer | Tools |
|---|---|
| Framework | Next.js 15 (App Router, Turbopack) |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| Database | SQLite + Prisma ORM |
| Auth | bcryptjs |
| Forms | react-hook-form + Zod |
| Animations | Framer Motion |
| Icons | Lucide React |
